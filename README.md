

### 💻 프로젝트 소개

+ React 와 Node.js를 이용하여 로그인과 회원가입 기능 및 CRUD로 SNS 프로젝트를 개발하였습니다.

+ 사진이나 글을 등록하여 일상을 기록하는 SNS 웹 사이트입니다.

### 🕓개발기간

+ 2024.10.24 ~ 2024.10.30
